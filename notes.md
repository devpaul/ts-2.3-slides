# TypeScript 2.3

* Language Service Extensibility
* Improved `tsc`
	* --init
	* --help
* Report errors in .js files
	* --checkJs
* A new `--strict` options
	* combines
		* --strictNullChecks
		* --noImplicitAny
		* --noImplicitThis
		* --alwaysStrict
* Improved JSX support
	* JSX children checking
	* JSX stateless component overload resolution
* Stronly typed `this` in object literals
* Generic Defaults
* Async iterators
	* --downlevelIteration
* Generator support for ES3/ES5
